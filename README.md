# VR Мини-игра "Сбор объектов"

## Описание проекта
Виртуальная игра на основе фреймворка A-Frame, созданная в рамках практической работы по дисциплине "Разработка виртуальной и дополненной реальности".

## Цель игры
Собрать все 10 объектов, разбросанных по виртуальной сцене, за 60 секунд.

## Управление
- **WASD** - перемещение по сцене
- **Мышь** - вращение камеры
- **Клик ЛКМ** - сбор объекта (или автоматический сбор при приближении)

## Особенности реализации
1. **Полноценная VR сцена** с небом и землёй
2. **10 различных 3D объектов** для сбора с уникальными анимациями
3. **Система подсчёта очков** и таймер
4. **Отслеживание столкновений** для автоматического сбора
5. **HUD интерфейс** с информацией об игре
6. **Экран завершения игры** с возможностью перезапуска
7. **Звуковые эффекты** при сборе объектов и фоновая музыка
8. **Загрузка 3D модели** в формате GLTF
9. **Адаптивный дизайн** для различных устройств

## Технологии
- HTML5
- JavaScript
- A-Frame 1.6.0
- Three.js (через A-Frame)
- CSS3

## Запуск игры
1. Откройте файл `VRgameKK.html` в любом современном браузере
2. Разрешите доступ к микрофону и гироскопу при запросе
3. Для VR режима используйте кнопку "Enter VR" в правом нижнем углу
